# Final Year Project

## Project Structure
- `frontend-react`: The React frontend application.
- `backend`: The Python backend API.
- `scripts`: Data processing scripts.

## Setup Instructions

### Frontend
The `node_modules` folder is excluded from this repository to save space. To install dependencies:

1. Open a terminal in `frontend-react/`.
2. Run:
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```

### Backend
1. Create a virtual environment:
   ```bash
   python -m venv venv
   ```
2. Activate it and install requirements:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the backend:
   ```bash
   python main.py
   ```
